#  El paquete {CDR}
![LogoCDR 1280x640](https://github.com/cdr-book/CDR/assets/80209018/0262e65c-d2ba-4b9b-bb1f-8fa81bbd34aa)

El libro Fundamentos de ciencia de datos con R dispone de un paquete en GitHub, {CDR} que contiene los conjuntos de datos utilizados en el libro y que se puede descargar con el siguiente c√≥digo:


`install.packages("remotes")`  
`remotes::install_github("cdr-book/CDR")`

La lista de todos los conjuntos de datos puede obtenerse haciendo `data()`.

`library("CDR")`  
`data(package = "CDR")`
